# Description
selenium 라이브러리를 이용해 네이버 항공권 웹 사이트의 데이터 스크래핑을 진행했다. 부산에서 제주까지 가는 항공편을 출발 시각이 빠른 순으로 상위 10개의 데이터를 가져와 엑셀 파일로 저장할 수 있다. 수집 데이터는 항공사, 출발 시각, 도착 시각, 소요 시간, 가격 등이 있다.

# Environment
- Python
- Selenium
- Chrom Browser
